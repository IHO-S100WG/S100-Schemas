S100_MD_LanguageCode :

Definition: Language codes
CodeSpace: http://www.iho.int/S100/lan/1.0
Description: Subset of Alpha-3 language codes according to ISO 639-2. The Registration Authority for the alpha-3 language code elements contained in ISO 639-2 is the Library of Congress (LoC). This file uses the Terminology Codes (639-2/T). The names are as in the register maintained by the Registation Authority as of the date this list was prepared, and are given in the definition tags. Definition: Data encoding format codes

Description: Data encoding format codes for coverage and pointset data described in S-100 Parts 8 and 10c.
Number of items: 9
Entry Definition Description
fixedStations Time series at fixed stations Data at multiple discrete fixed point locations
regularGrid Regularly-gridded arrays Data at grid points forming a regular grid with constant cell spacing
ungeorectifiedGrid Ungeorectified gridded arrays Data that does not include any information that can be used to determine a cell’s geographic coordinate values, or in which cell spacing is variable, and there is no predefined association between one cell’s location and that of another
movingPlatform Moving Platform Data at sequential discrete point locations of a moving sensor platform
irregularGrid Irregularly-gridded data; Data distributed over a grid with uniform cell spacing but irregular overall shape
variableCellSize Grids with variable cell sizes Variable-density grid containing one or more regions with cell spacing that is a whole multiple of a common minimum uniform cell spacing
TIN Triangulated irregular network Triangulated irregular network. A TIN is a representation of a continuous surface consisting entirely of triangular facets.
